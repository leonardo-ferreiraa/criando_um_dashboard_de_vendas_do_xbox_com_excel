# Criando_um_dashboard_de_vendas_do_xbox_com_excel

Projeto feito na DIO em parceria com Trainee Santander.
A proposta deste projeto é **criar um dashboard de vendas e visualização de dados** básico.

O objetivo é transformar dados brutos em **informações visuais claras e úteis**, permitindo a análise eficaz do desempenho de vendas e a tomada de decisões baseadas em dados.

- 1º Gráfico - Mostra o total de vendas do Game Pass do EA Play

- 2º Gráfico - Mostra o total de vendas do Game Pass do Minecraft

- 3º Gráfico - Mostra faturamento Total de venda em planos anuais, separando por auto renovação não é por auto renovação
